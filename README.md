# 前言

随着科技的发展，智慧医疗逐渐成为医疗行业的热点。本项目是基于SSM（Spring+SpringMVC+MyBatis）框架开发的智慧医疗系统，致力于为用户提供便捷、高效的医疗服务。

# 内容介绍

本智慧医疗系统主要包括以下功能模块：患者管理、医生管理、预约挂号、就诊记录管理、药品管理等。通过这些模块，实现患者与医生之间的在线沟通、预约挂号、查看就诊记录等操作，提高医疗服务的效率和质量。
